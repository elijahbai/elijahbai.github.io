### flask

```shell
flask是python下的一个框架，与Django相比，它小型化，更灵活，可扩展性更强。
用flask写一个简单的web程序，如下：
```

```python
from flask import Flask 
app = Flask(__name__) 

@app.route("/")
def hello(): 
    return "Welcome to flask!" 

if __name__ == "__main__":
    app.run()
```
